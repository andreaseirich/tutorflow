# PWA Installation Guide

TutorFlow is available as a Progressive Web App (PWA), which means you can install it on your device and use it like a native app. This guide explains how to install TutorFlow on different platforms.

## What is a PWA?

A Progressive Web App (PWA) combines the best features of web and mobile apps:
- **Installable**: Add it to your home screen
- **Offline Support**: Works even without internet connection (cached content)
- **Fast**: Quick loading and smooth performance
- **App-like Experience**: Full-screen mode, no browser UI
- **Always Up-to-date**: Automatically updates when new versions are available

## Installation Instructions

### Desktop (Chrome, Edge, Brave)

1. **Open TutorFlow** in your browser
2. **Look for the install icon** in the address bar (usually appears as a "+" or install icon)
3. **Click the install icon** or look for "Install TutorFlow" in the browser menu
4. **Confirm installation** when prompted
5. TutorFlow will now appear in your applications list and can be launched like a regular app

**Alternative method:**
- Click the menu (three dots) in the top-right corner
- Select "Install TutorFlow" or "Install app"
- Confirm the installation

### Desktop (Firefox)

Firefox has limited PWA support. You can:
1. Use the browser's "Add to Home Screen" feature if available
2. Create a bookmark and use it as a shortcut
3. Consider using Chrome, Edge, or Brave for full PWA support

### iOS (iPhone/iPad)

1. **Open TutorFlow** in Safari (PWA features require Safari)
2. **Tap the Share button** (square with arrow pointing up) at the bottom
3. **Scroll down** and tap **"Add to Home Screen"**
4. **Edit the name** if desired (default: "TutorFlow")
5. **Tap "Add"** in the top-right corner
6. TutorFlow icon will appear on your home screen
7. Tap the icon to launch TutorFlow in full-screen mode

**Note:** On iOS, PWAs work best in Safari. Other browsers may have limited support.

### Android (Chrome)

1. **Open TutorFlow** in Chrome
2. **Look for the install banner** at the bottom of the screen
3. **Tap "Install"** or "Add to Home Screen"
4. **Confirm installation** when prompted
5. TutorFlow icon will appear on your home screen
6. Tap the icon to launch TutorFlow in app mode

**Alternative method:**
- Tap the menu (three dots) in the top-right corner
- Select "Add to Home screen" or "Install app"
- Confirm the installation

### Android (Other Browsers)

Most modern Android browsers support PWAs:
- **Firefox**: Menu → "Add to Home Screen"
- **Samsung Internet**: Menu → "Add page to"
- **Opera**: Menu → "Add to Home screen"

## After Installation

Once installed, TutorFlow will:
- Open in its own window (no browser UI)
- Remember your login session
- Work offline for cached pages
- Update automatically when new versions are available
- Appear in your device's app list

## Uninstalling

### Desktop
- Right-click the TutorFlow icon in your applications
- Select "Uninstall" or "Remove"
- Or use your system's application manager

### Mobile
- Long-press the TutorFlow icon
- Select "Uninstall" or "Remove"
- Or use your device's app manager

## Troubleshooting

### Install option not appearing

**Possible reasons:**
- Your browser doesn't support PWAs (use Chrome, Edge, or Safari)
- The site is not served over HTTPS (required for PWAs)
- Service Worker registration failed (check browser console)

**Solutions:**
- Update your browser to the latest version
- Ensure you're accessing TutorFlow over HTTPS
- Clear browser cache and try again
- Check browser console for errors

### Offline mode not working

- Some features require an internet connection
- Only previously visited pages are available offline
- Clear cache and revisit pages to update offline content

### App not updating

- PWAs update automatically when new versions are available
- You may need to close and reopen the app
- Clear cache if updates don't appear

## Features

### Offline Support
- Previously visited pages are cached and available offline
- Service Worker handles caching automatically
- Some features may require internet connection

### Push Notifications
- Notifications can be enabled (if implemented)
- Requires permission from the user
- Works even when the app is closed

### App Shortcuts
- Quick access to Dashboard and Calendar
- Available from the app icon context menu (long-press)

## System Requirements

- **Desktop**: Modern browser (Chrome 73+, Edge 79+, Firefox 90+, Safari 11.1+)
- **iOS**: iOS 11.3+ with Safari
- **Android**: Android 5.0+ with Chrome or other modern browser
- **Internet**: Required for initial setup and real-time features

## Privacy and Security

- PWAs use the same security model as regular websites
- Data is stored locally on your device
- HTTPS encryption ensures secure communication
- No additional permissions required beyond what the website needs

## Support

If you encounter issues installing or using TutorFlow as a PWA:
1. Check that your browser supports PWAs
2. Ensure you're using the latest browser version
3. Clear browser cache and cookies
4. Try a different browser
5. Contact support if problems persist
